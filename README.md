# זרם — השוואת תוכניות חשמל

אתר סטטי להשוואת תוכניות חשמל בישראל לפי קובץ פעימות המונה של המשתמש. העיבוד מתבצע בדפדפן בלבד; קובץ הצריכה אינו נשלח לשרת.

## הרצה מקומית

```bash
pnpm dev
```

פתחו את [http://localhost:3000](http://localhost:3000).

## בדיקות ובנייה

```bash
pnpm test
pnpm lint
pnpm build
```

הבנייה מפיקה static export לתיקייה `out/` ומתאימה ל-GitHub Pages.

## פרטיות

CSV, PDF, תיקיות פלט וקבצים זמניים חסומים ב-`.gitignore` ואינם אמורים להיכלל במאגר.
נתוני ההדגמה הכלולים במאגר הם סינתטיים ואינם מבוססים על קובץ צריכה אישי.
